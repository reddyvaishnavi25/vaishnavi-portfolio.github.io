Movie Review Prediction using Real Time Twitter Data

Tech Stack: Python, NLP, Vader, Affin, CNN, RNN, Bert, RoBERT

Github URL: Project Link

  • Collect around 100,000 tweets related to popular movies using the Twitter API.
  • Preprocess the collected tweets by removing noise like special characters, URLs, and emojis, and tokenize the text.
  • Utilize sentiment analysis models including Vader, Affin, CNN, RNN, BERT, and RoBERTa.
  • Train each model on labeled movie review datasets and optimize hyperparameters.
  • Evaluate model performance using metrics such as accuracy, precision, recall, and F1-score.
